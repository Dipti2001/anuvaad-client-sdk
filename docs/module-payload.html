payload

Description:

  • This module provides functions to generate payloads for various translation function. A payload is a structured data object that contains the necessary input data and configuration parameters required to perform different translation tasks such as Automatic Speech Recognition (ASR), Neural Machine Translation (NMT), and Text-to-Speech (TTS). These functions assist in constructing payloads that can be sent to the server for processing.
